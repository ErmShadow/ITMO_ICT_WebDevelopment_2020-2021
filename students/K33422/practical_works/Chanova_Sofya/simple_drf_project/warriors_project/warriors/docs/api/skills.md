# show all skills

displays all skills info

**URL** : `skills/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

**Success info** : HTTP 200 OK

**Content-Type** : application/json

**JSON** :
```json
{
    "Skills": [
        {
            "title": "swimming"
        },
        {
            "title": "fighting"
        }
    ]
}
```
