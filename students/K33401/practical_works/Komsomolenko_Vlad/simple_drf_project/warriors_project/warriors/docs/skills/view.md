# Вывести список всех скиллов

Вывод всех имеющихся скиллов.

**URL** : `war/skills/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

**Success info** : HTTP 200 OK

**Content-Type** : application/json

**JSON** :
```json
{
    "Skill": [
        {
            "title": "kill"
        }
    ]
}
```
